About

Ayase Quart (AQ) is an interface for 4chan/Lainchan archives using the Asagi database schema. What sets it apart from other archives is its advanced search options such as,
  • - Searching replies with certain OPs - good for searching generals.
  • - Full-text-search (FTS) using lnx-search. Meilisearch, and others engines are partially supported.
  • - Gallery mode for search results.
  • - Written using Python3.13 and Quart.
AQ supports MySQL and SQLite, so it's compatible with Neofuuka (MySQL), Hayden (MySQL), and Ritual (SQLite) archive downloaders.
